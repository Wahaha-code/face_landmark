# Face Detection Project

This project is a real-time,high accuracy face detection. The main script of the project is `main.py`.

## Requirements

- Python 3.8 or later
- PyTorch 1.8.1 or later
- torchvision 0.9.1 or later
- OpenCV 4.5.2 or later
